Bonjour [NOM_PARENT],

- Élève: [PRENOM_ELEVE]
- Matière/niveau: [NIVEAU + MATIERE]
- Tuteur: [NOM_TUTEUR]
- Date et heure: [DATE + HEURE]
- Format: [EN LIGNE / PRESENTIEL]

Vous recevrez aussi l'invitation calendrier.

Pour modifier ou reporter cette séance, prévenez-nous au moins 72 h à l'avance : le report est garanti. En cas d'imprévu sous 72 h, écrivez-nous; l'équipe cherchera une solution selon les disponibilités, sans perte automatique de paiement ou de crédit.

Avant la séance, vous pouvez envoyer une note de préparation dans le portail : le chapitre, devoir, examen ou résultat qui inquiète le plus. Cela aide le tuteur à cibler rapidement. Les photos et fichiers ne se téléversent pas encore dans le portail; si un document est indispensable, nous conviendrons d'un canal sécurisé avec vous.

Merci,
